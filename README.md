## English

### Professional Theme Collection for JetBrains IDEs

JBL Themes is a premium theme plugin that brings 13 carefully crafted color schemes to all JetBrains IDEs, including IntelliJ IDEA, PyCharm, WebStorm, PhpStorm, and more.

### ✨ Features

- 🎨 **13 Beautiful Themes** - 4 free themes + 9 premium themes
- 🖼️ **Custom Backgrounds** - Each theme with carefully designed background images
- ✨ **Dynamic Effects** - Cherry blossom petals, snowflakes, neon rings, and matrix rain
- 🎯 **Zero Flicker** - Optimized rendering for smooth visual experience
- ⚙️ **Customizable** - Adjust background opacity and use your own images
- 🌍 **Universal Support** - Works with all JetBrains IDEs

### 📦 Themes

#### Free Themes (4)

- **JBL Light** - Clean and simple light theme, perfect for daily use
- **JBL Light Warm** - Eye-comfort warm tones for long coding sessions
- **JBL Eye Friendly** - Eye-friendly green background with optimized font settings
- **JBL Dark** - Classic dark theme with comfortable contrast

#### Premium Themes (9) - $5

**Static Background Themes:**
- **JBL Light Cool** - Tech-inspired cool white
- **JBL Light Beige** - Professional business style
- **JBL Light Pearl** - Soft pearl luster
- **JBL Light Ivory** - Vintage ivory paper
- **JBL Light Paper** - Natural paper texture

**Dynamic Effect Themes:**
- **JBL Light Snow** - Crystal clear snow white with snowflake effects ❄️
- **JBL Cherry Blossom** - Romantic sakura pink with falling petal effects 🌸
- **JBL Dark Neon** - Cyberpunk neon style with glowing ring effects 💫
- **JBL Dark Matrix** - Hacker Matrix style with code rain effects 💻

### 🎮 Customization

Navigate to `Settings > Appearance > JBL Themes` to customize:

- **Background Opacity** - Adjust transparency from 0% to 100%
- **Custom Background Image** - Use your own background images (PNG, JPG, JPEG)

### 🚀 Installation

1. Open your JetBrains IDE
2. Go to `Settings > Plugins`
3. Search for "JBL Themes"
4. Click `Install`
5. Restart your IDE
6. Go to `Settings > Appearance & Behavior > Appearance`
7. Select your desired theme from the dropdown

## 中文说明

### JetBrains IDE 专业主题合集

JBL Themes 是一款为所有 JetBrains IDE（包括 IntelliJ IDEA、PyCharm、WebStorm、PhpStorm 等）精心打造的高级主题插件，提供 13 种精美配色方案。

### ✨ 特性

- 🎨 **13 种精美主题** - 4 款免费主题 + 9 款高级主题
- 🖼️ **自定义背景** - 每个主题都配有精心设计的背景图片
- ✨ **动态特效** - 樱花花瓣、雪花飘落、霓虹光圈、代码雨等特效
- 🎯 **零闪烁** - 优化的渲染机制，流畅的视觉体验
- ⚙️ **可自定义** - 调整背景透明度，使用自己的背景图片
- 🌍 **通用支持** - 兼容所有 JetBrains IDE

### 📦 主题列表

#### 免费主题（4 款）

- **JBL Light** - 简洁明快的浅色主题，适合日常使用
- **JBL Light Warm** - 护眼暖色调，适合长时间编码
- **JBL Eye Friendly** - 护眼绿色背景，优化的字体设置
- **JBL Dark** - 经典深色主题，舒适的对比度

#### 高级主题（9 款）- ¥5

**静态背景主题：**
- **JBL Light Cool** - 科技感冷白色调
- **JBL Light Beige** - 专业商务风格
- **JBL Light Pearl** - 柔和珍珠光泽
- **JBL Light Ivory** - 复古象牙纸质感
- **JBL Light Paper** - 自然纸张纹理

**动态特效主题：**
- **JBL Light Snow** - 晶莹剔透雪白色 + 雪花飘落特效 ❄️
- **JBL Cherry Blossom** - 浪漫樱花粉 + 花瓣飘落特效 🌸
- **JBL Dark Neon** - 赛博朋克霓虹风 + 光圈特效 💫
- **JBL Dark Matrix** - 黑客矩阵风 + 代码雨特效 💻

### 🎮 自定义设置

在 `Settings > Appearance > JBL Themes` 中自定义：

- **背景透明度** - 从 0% 到 100% 调节透明度
- **自定义背景图片** - 使用自己的背景图片（支持 PNG、JPG、JPEG）
